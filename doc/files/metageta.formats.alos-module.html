Module alos

source code

Metadata driver for ACRES ALOS AVNIR-2/PRISM/PALSAR imagery

Supports:

  • ALOS AVNIR2/PRISM
  • ALOS PALSAR (Level 1.5 only, Level 1.0 not (yet?) implemented)

Format specifications:

  • PALSAR Level 1.0: http://www.ga.gov.au/servlet/BigObjFileManager?bigobjid=GA10287
  • PALSAR Level 1.1/1.5: http://www.eorc.jaxa.jp/ALOS/doc/fdata/PALSAR_x_Format_EK.pdf
  • ALOS AVNIR2/PRISM: http://www.ga.gov.au/servlet/BigObjFileManager?bigobjid=GA10285
