Module runcrawler

source code

Script to run the MetaGeta Metadata Crawler

Contains code to show GUI to gather input arguments when none are provided To run, call the eponymous batch file/shell script which sets the required environment variables

Usage:

   runcrawler.bat/sh -d dir -x xlsx {-o} {--debug}
Arguments:
  • -d [dir] : Directory to search for imagery
  • -x [xlsx] : MS Excel 2007 spreadsheet to write metadata to
  • -m, --mediaid : CD/DVD media ID, defaults to volume label.
  • -u, --update : Update previous crawl results.
  • -o, --overviews : Generate overview (quicklook/thumbnail) images")
  • -r, --recurse : Search directory recursively?
  • -a, --archive : Search compressed archives?
  • --debug : Turn debug output on

Note: See Issue 22
