Metadata Transforms

Script to run the Metadata Transforms.

Contains code to show GUI to gather input arguments when none are provided. To run, call the eponymous batch file which sets the required environment variables.

Usage:

   runtransform.bat -x xlsx -t xsl -d dir
Arguments:
  • -x xlsx: MS Excel 2007 spreadsheet to read from
  • -t xsl: XSL transform - may be one of the pre-defined XSL transforms or a path to a custom XSL file.
  • -d dir: Directory to write XML files to.

Note: Additional metadata elements

The ANZLIC ISO19139 stylesheet can make use of additional metadata elements that may be manually added to the Excel spreadsheet and will be included in the output XML/MEF metadata. See the transforms module documentation for more information.
